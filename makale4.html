Windows 8 – 0x80248014 Hatası
Windows Tuşu + R Basın ve “services.msc” yazın.

En alttaki Windows Update'e sağ tıklayıp durdura tıklayın. Tekrar Windows Tuşu + R basıp ”%windir%” yazıp tamama basın.

”SoftwareDistribution” adlı klasörü silin. Sildikten sonra Windows tuşunu basın ve cmd yazıp enterlayın.

Cmd komutuna ”net start wuauserv” yazıp enterlayın. Bunları yaptıktan sonra sorun düzelecektir.
